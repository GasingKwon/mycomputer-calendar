# Decisions

이 문서는 제품/기술 결정 사항을 기록합니다.

## 2026-06-25

### Product Direction

- 앱 이름은 **할 일을 기억해라**로 결정했습니다.
- 기본 화면은 월간 달력형으로 결정했습니다.
- 오른쪽 상세 패널에서 날짜별 할 일을 관리합니다.
- 구글 캘린더와 비슷한 깔끔한 디자인을 지향합니다.

### Task Model

- 할 일 추가는 각각 입력하는 폼 방식으로 결정했습니다.
- 날짜칸 미리보기는 최대 3개 표시 후 `+N개`로 표현합니다.
- 완료된 할 일은 회색 글자와 취소선으로 표시합니다.
- 중요도는 `낮음`, `보통`, `높음` 3단계로 결정했습니다.
- 반복 일정은 `N일마다 반복` 방식으로 결정했습니다.

### Desktop Behavior

- 창 크기 자유 조절을 지원합니다.
- 창 위치를 기억합니다.
- 항상 위에 고정을 지원합니다.
- 투명도 조절을 지원합니다.
- 트레이 숨기기를 지원합니다.
- Windows 시작 시 자동 실행은 옵션으로 제공합니다.

### Technology

- Electron + React + TypeScript를 사용합니다.
- 데이터는 로컬 JSON 파일로 시작합니다.
- 데이터 위치는 사용자 AppData 폴더로 계획합니다.
- 알림은 Windows 기본 알림과 기본 알림음을 사용합니다.
