# UX Flow

## 1. App Launch

1. 사용자가 앱을 실행합니다.
2. 앱은 로컬 JSON 파일을 불러옵니다.
3. 저장된 창 위치, 크기, 투명도, 항상 위 설정을 복원합니다.
4. 현재 월의 달력을 표시합니다.
5. 오늘 날짜가 기본 선택됩니다.

## 2. Month Navigation

1. 사용자가 이전/다음 버튼을 누릅니다.
2. 달력은 해당 월로 이동합니다.
3. 오른쪽 패널은 선택된 날짜를 유지하되, 선택 날짜가 현재 표시 월 밖이면 오늘 또는 해당 월 1일로 이동할 수 있습니다.

## 3. Select Date

1. 사용자가 날짜칸을 클릭합니다.
2. 선택 강조가 해당 날짜로 이동합니다.
3. 오른쪽 패널에 해당 날짜의 할 일이 표시됩니다.

## 4. Add Task

1. 사용자가 오른쪽 패널의 추가 폼에 제목을 입력합니다.
2. 필요하면 시간, 중요도, 마감일, 반복 간격, 알림 여부, 메모를 설정합니다.
3. 저장 버튼을 누릅니다.
4. 새 할 일이 선택 날짜에 추가됩니다.
5. JSON 파일에 저장됩니다.

## 5. Complete Task

1. 사용자가 할 일 체크박스를 클릭합니다.
2. 완료 상태가 토글됩니다.
3. 완료된 항목은 회색 글자와 취소선으로 표시됩니다.
4. 완료된 항목은 알림 대상에서 제외됩니다.
5. JSON 파일에 저장됩니다.

## 6. Edit Task

1. 사용자가 할 일을 선택합니다.
2. 폼에 기존 값이 채워집니다.
3. 사용자가 내용을 수정하고 저장합니다.
4. 달력 미리보기와 상세 목록이 갱신됩니다.
5. JSON 파일에 저장됩니다.

## 7. Delete Task

1. 사용자가 삭제 버튼을 누릅니다.
2. 앱은 삭제 확인을 요청합니다.
3. 확인하면 항목을 삭제합니다.
4. JSON 파일에 저장됩니다.

## 8. Tray Hide

1. 사용자가 트레이 숨기기를 누르거나 창 닫기 동작을 수행합니다.
2. 앱은 완전히 종료되지 않고 트레이로 숨겨집니다.
3. 트레이 아이콘을 클릭하면 창이 다시 표시됩니다.

## 9. Settings

설정에서 다음 항목을 조작할 수 있습니다.

- 항상 위에 고정
- 투명도
- Windows 시작 시 자동 실행

변경 사항은 즉시 적용하고 JSON 파일에 저장합니다.
