# Product Spec: 할 일을 기억해라

## 1. Overview

**할 일을 기억해라**는 Windows에서 실행되는 로컬 전용 캘린더형 할 일 앱입니다.

사용자는 월간 달력에서 날짜별 할 일을 확인하고, 오른쪽 상세 패널에서 할 일을 추가, 수정, 완료, 삭제할 수 있습니다. 앱은 구글 캘린더와 비슷한 깔끔한 디자인을 지향하되, 윈도우 스티커 메모처럼 가볍게 띄워두는 사용감을 제공합니다.

## 2. Target User

- 개인 PC에서 해야 할 일을 날짜별로 관리하고 싶은 사용자
- 복잡한 계정, 동기화, 협업 기능 없이 로컬에서만 쓰고 싶은 사용자
- 바탕화면 근처에 작은 달력/메모처럼 띄워두고 싶은 사용자

## 3. Product Goals

- 월간 달력 기반의 빠른 일정 확인
- 날짜별 할 일의 쉬운 추가/수정/삭제
- 시간 일정, 중요도, 마감일, 반복 일정, 알림 지원
- 2000ml 일일 물 마시기 기록과 달성/미달성 표시
- Windows 데스크톱 환경에 맞는 창 제어
- 계정 없이 로컬 파일만으로 데이터 보관
- AI 구현과 검증이 쉬운 명세/하네스 구조 유지

## 4. Non-Goals for MVP

- 클라우드 동기화
- 모바일 앱
- 여러 사용자/공유 캘린더
- Google Calendar 연동
- 자연어 일정 입력
- 복잡한 반복 규칙
- 태그/프로젝트/칸반 보드

## 5. Core Layout

기본 화면은 **월간 달력 + 오른쪽 상세 패널**입니다.

### Left: Month Calendar

- 현재 월 전체를 표시합니다.
- 이전 달/다음 달로 이동할 수 있습니다.
- 오늘 날짜를 강조합니다.
- 선택한 날짜를 강조합니다.
- 날짜칸에는 할 일을 최대 3개까지 미리보기로 표시합니다.
- 3개를 초과하면 `+N개` 형식으로 추가 개수를 표시합니다.
- 완료된 할 일은 회색 글자와 취소선으로 표시합니다.
- 중요도 높은 항목은 더 눈에 띄는 색상으로 표시합니다.

### Right: Day Detail Panel

- 선택한 날짜를 표시합니다.
- 해당 날짜의 전체 할 일 목록을 표시합니다.
- 할 일 추가 폼을 제공합니다.
- 기존 할 일을 수정/삭제할 수 있습니다.
- 완료 체크를 할 수 있습니다.
- 시간, 중요도, 마감일, 반복, 알림, 메모를 설정할 수 있습니다.

## 6. Task Capabilities

할 일은 다음 정보를 가질 수 있습니다.

- 제목
- 날짜
- 완료 여부
- 시간, 선택 사항
- 중요도: 낮음, 보통, 높음
- 카테고리: 사용자 정의 가능
- 마감일, 선택 사항
- 반복: 없음 또는 N일마다 반복
- 알림 켜기/끄기
- 메모, 선택 사항

## 7. Recurrence

반복 일정은 MVP에서 **N일마다 반복**만 지원합니다.

예시:

- 1일마다 반복: 매일
- 7일마다 반복: 매주
- 30일마다 반복: 한 달 주기처럼 사용
- 365일마다 반복: 매년 주기처럼 사용

반복 항목은 원본 날짜부터 계산하여 선택한 날짜에 발생 여부를 판정합니다.

## 8. Notifications

Windows 기본 알림과 기본 알림음을 사용합니다.

알림 대상:

- 시간이 있는 일정
- 마감일이 있는 할 일
- 중요도 높음 항목
- 반복 일정의 발생일

알림 제외:

- 완료된 할 일
- 알림이 꺼진 할 일

## 9. Window Behavior

앱은 스티커 메모처럼 사용할 수 있어야 합니다.

- 창 크기 자유 조절
- 창 위치 기억
- 항상 위에 고정
- 투명도 조절
- 트레이 숨기기
- Windows 시작 시 자동 실행 옵션

## 10. Storage

- 계정 없음
- 서버 없음
- 인터넷 연결 불필요
- 사용자 AppData 폴더에 JSON 파일로 저장
- 추후 백업/내보내기 기능 확장 가능

## 11. Design Direction

구글 캘린더와 비슷한 깔끔하고 정보 중심적인 디자인을 지향합니다.

- 흰색/연회색 기반
- 선명하지만 과하지 않은 날짜칸
- 일정은 작은 컬러 라벨처럼 표시
- 중요도는 색상으로 구분
- 버튼과 입력창은 단순하고 현대적으로 구성
- 과한 장식보다 가독성과 빠른 조작을 우선

## 12. MVP Acceptance Summary

MVP는 사용자가 앱을 실행해 월간 달력에서 날짜별 할 일을 관리하고, 로컬 저장 및 기본 Windows 창 기능을 사용할 수 있을 때 완료로 봅니다. 상세 기준은 `harness/checklists/mvp-acceptance.md`를 따릅니다.
