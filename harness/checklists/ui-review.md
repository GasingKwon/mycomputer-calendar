# UI Review Checklist

- [ ] 월간 달력이 첫 화면에서 명확하게 보인다.
- [ ] 오른쪽 상세 패널은 선택 날짜와 연결되어 보인다.
- [ ] 입력 폼은 너무 복잡하지 않다.
- [ ] 중요도 색상이 과하지 않으면서 구분된다.
- [ ] 완료 상태가 회색과 취소선으로 명확히 보인다.
- [ ] 날짜칸의 할 일 미리보기 텍스트가 넘치지 않는다.
- [ ] `+N개` 표시가 자연스럽다.
- [ ] 창 크기를 줄여도 주요 조작이 깨지지 않는다.
- [ ] 투명도 적용 후에도 텍스트 가독성이 유지된다.
- [ ] 구글 캘린더 계열의 깔끔한 인상이 유지된다.
