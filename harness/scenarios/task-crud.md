# Scenario: Task CRUD

## Goal

사용자가 날짜별 할 일을 추가, 확인, 수정, 완료, 삭제할 수 있어야 합니다.

## Preconditions

- 앱이 실행되어 있습니다.
- 오늘 날짜가 선택되어 있습니다.
- 데이터 파일이 정상적으로 로드되었습니다.

## Scenario A: Add Task

1. 오른쪽 상세 패널에서 제목을 입력합니다.
2. 중요도를 선택합니다.
3. 필요하면 시간과 마감일을 입력합니다.
4. 저장 버튼을 누릅니다.

Expected:

- 선택 날짜의 상세 목록에 새 할 일이 표시됩니다.
- 월간 달력 날짜칸 미리보기에 새 할 일이 표시됩니다.
- 데이터 파일에 새 할 일이 저장됩니다.

## Scenario B: Edit Task

1. 기존 할 일을 선택합니다.
2. 제목 또는 중요도를 수정합니다.
3. 저장 버튼을 누릅니다.

Expected:

- 상세 목록과 달력 미리보기가 수정된 값으로 갱신됩니다.
- `updatedAt`이 변경됩니다.
- 데이터 파일에 변경 사항이 저장됩니다.

## Scenario C: Complete Task

1. 할 일 체크박스를 클릭합니다.

Expected:

- 항목이 완료 상태로 변경됩니다.
- 텍스트가 회색과 취소선으로 표시됩니다.
- 완료 항목은 알림 대상에서 제외됩니다.
- 데이터 파일에 완료 상태가 저장됩니다.

## Scenario D: Delete Task

1. 할 일 삭제 버튼을 누릅니다.
2. 삭제 확인에 동의합니다.

Expected:

- 항목이 상세 목록에서 제거됩니다.
- 날짜칸 미리보기에서도 제거됩니다.
- 데이터 파일에서도 제거됩니다.
